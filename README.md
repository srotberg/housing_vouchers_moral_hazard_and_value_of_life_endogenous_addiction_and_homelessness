Replication code for "Housing Vouchers, Moral Hazard, and Value of Life: Endogenous Addiction and Homelessness" by Shahar Rotberg

The "code" folder contains the Fortran code that runs the quantitative experiments. Please see the text file "readme.txt" in this folder for detailed instructions on how to compile and run the code.

The "data" folder contains and codes used to analyze the data.

The "outputs" folder includes all the simulations I ran.
